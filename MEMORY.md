# VersionCheck Memory 🧠

## v0.1.8 (2026-05-24)
- **Antigravity CLI (`agy`)**: Добавлена проверка версий через официальный эндпоинт авто-апдейтера и автоматическое обновление через `agy update`.
- **Antigravity IDE (`agy-ide`)**: Интегрирована проверка версии. Локальная версия считывается из файла `product.json` (`ideVersion`), удаленная — с угаданного нами API авто-апдейтера. Выставлен режим `ignore_update=True` для ручного обновления на Linux.
- **OpenCode**: Добавлен мониторинг версий из репозитория `anomalyco/opencode` с поддержкой автообновления через `opencode upgrade`.
- **Универсальные URL для версий (`version_url`)**: Внедрена поддержка произвольных веб-эндпоинтов для проверки актуальных версий софта в обход GitHub Releases.
- **Dynamic Paths**: Скрипт переведен на динамическое определение собственного пути через `Path(__file__).resolve()`, что делает его полностью независимым от текущего рабочего каталога запуска.
- **Aesthetics & Alignment**: Вся сетка вывода терминала расширена с 12 до 15 символов ширины имени для идеального вертикального выравнивания двоеточий.
